This guide walks you through creating your first PlayStation ISO image with mkpsxiso.

## Prerequisites

<Steps>
  <Step title="Install mkpsxiso">
    Make sure you have mkpsxiso installed. See the [Installation](/installation) guide if you haven't installed it yet.
  </Step>
  
  <Step title="Prepare your files">
    Create a project directory with your PlayStation executable and assets:
    
    ```bash
    mkdir my-ps1-game
    cd my-ps1-game
    ```
  </Step>
</Steps>

## Create an XML project file

Create a file named `project.xml` with your ISO project configuration:

<CodeGroup>

```xml Minimal example
<?xml version="1.0" encoding="UTF-8"?>

<iso_project image_name="game.bin" cue_sheet="game.cue">
  <track type="data">
    <identifiers
      system="PLAYSTATION"
      application="PLAYSTATION"
      volume="MYGAME"
      publisher="MYNAME"
    />
    
    <directory_tree>
      <file name="system.cnf" type="data" source="system.txt"/>
      <file name="game.exe" type="data" source="game.exe"/>
      
      <dir name="data">
        <file name="level1.dat" type="data" source="data/level1.dat"/>
      </dir>
    </directory_tree>
  </track>
</iso_project>
```

```xml With license
<?xml version="1.0" encoding="UTF-8"?>

<iso_project image_name="game.bin" cue_sheet="game.cue">
  <track type="data">
    <identifiers
      system="PLAYSTATION"
      application="PLAYSTATION"
      volume="MYGAME"
      publisher="MYNAME"
    />
    
    <license file="licensea.dat"/>
    
    <directory_tree>
      <file name="system.cnf" type="data" source="system.txt"/>
      <file name="game.exe" type="data" source="game.exe"/>
      
      <dir name="data">
        <file name="level1.dat" type="data" source="data/level1.dat"/>
      </dir>
    </directory_tree>
  </track>
</iso_project>
```

</CodeGroup>

## Create a system.cnf file

Create `system.txt` with your boot configuration:

```text
BOOT = cdrom:\GAME.EXE;1
TCB = 4
EVENT = 10
STACK = 801FFFF0
```

<Note>
The `BOOT` line specifies which executable to run. The `;1` suffix is required by the ISO9660 standard.
</Note>

## Build the ISO

Run mkpsxiso to build your ISO image:

```bash
mkpsxiso project.xml
```

You should see output like:

```text
MKPSXISO 2.20 - PlayStation ISO Image Maker
...
ISO image generated successfully.
```

This creates two files:
- `game.bin` - The ISO image data
- `game.cue` - The cue sheet for CD burning/emulation

## Test your ISO

<Tabs>
  <Tab title="Emulator">
    Load the `.cue` file in an emulator like:
    - DuckStation
    - PCSX-Redux
    - ePSXe
    - Mednafen
    
    The emulator will use the `.cue` file to locate and load the `.bin` image.
  </Tab>
  
  <Tab title="Real Hardware">
    Burn the image to a CD-R:
    
    1. Use CD burning software that supports cue/bin format:
       - ImgBurn (Windows)
       - cdrdao (Linux)
       - Disc Utility (macOS with third-party tools)
    
    2. Select "Burn Image" and choose the `.cue` file
    
    3. Test on a modded PlayStation console
  </Tab>
</Tabs>

## Common options

### Generate LBA log

Create a log of where each file is located on the disc:

```bash
mkpsxiso project.xml -lba filelist.txt
```

### Override output name

Specify a different output filename:

```bash
mkpsxiso project.xml -o custom-name.bin
```

### Quiet mode

Suppress progress output:

```bash
mkpsxiso project.xml -q
```

<Warning>
For a bootable PlayStation disc, you need a license file. See the [License Data](/guides/license-data) guide for details.
</Warning>
